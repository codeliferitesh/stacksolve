# StackSolve 🚀

StackSolve is a modern anonymous college community platform built for students to share ideas, ask questions, discuss problems, and help each other in a safe and clean environment.

Built with Next.js 15, Firebase, Tailwind CSS, and TypeScript.

---

## ✨ Features

- 🔐 Secure Authentication
- 👤 Anonymous Usernames
- 📝 Create & Share Posts
- 💬 Nested Comments & Replies
- ❤️ Like & Bookmark System
- 🔔 Real-time Notifications
- 🔎 Explore & Search Posts
- 📱 Fully Responsive UI
- 🌙 Modern Clean Design
- 🛡️ Firebase Security Rules
- ⚡ Fast Performance
- ☁️ Firebase Backend

---

## 🛠️ Tech Stack

### Frontend
- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Zustand

### Backend & Services
- Firebase Authentication
- Firestore Database
- Firebase Storage
